# 車両予約管理システム

会社や病院の車両貸し出しを管理するためのシンプルなWebアプリケーションです。

## 機能

### 基本機能
- 複数車両の管理
- 週単位のカレンダー表示
- 時間単位での予約状況確認
- 予約の追加・削除
- 予約の重複チェック

### 予約情報
- 利用者名
- 部署・診療科
- 予約日時（開始時刻・終了時刻）
- 利用目的

## 使い方

1. **車両選択**: ドロップダウンから予約したい車両を選択
2. **予約確認**: カレンダーで空き状況を確認
3. **新規予約**: フォームに必要事項を入力して予約
4. **予約管理**: 予約一覧から既存の予約を確認・削除

## 技術仕様

- **フロントエンド**: HTML, CSS, JavaScript (Vanilla)
- **データ保存**: LocalStorage
- **レスポンシブ対応**: モバイル・タブレット・デスクトップ

## セットアップ

1. リポジトリをクローン
```bash
git clone https://github.com/mikito58/car-reservation-system.git
```

2. `index.html`をブラウザで開く
